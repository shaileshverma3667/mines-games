# Mines Game

A modern, interactive Mines game built with React and Vite. Similar to popular crypto gaming platforms, this game features manual and automatic gameplay modes with real-time multiplier calculations.

## Features

- **Manual Mode**: Click tiles to reveal gems while avoiding mines
- **Auto Mode**: Automated gameplay with customizable stop conditions
- **Dynamic Multipliers**: Real-time calculation based on revealed tiles
- **Betting System**: Place bets with demo mode support
- **Game Statistics**: Track wins, losses, and win rates
- **Game History**: View recent game results
- **Responsive Design**: Works on desktop and mobile devices

## How to Play

1. Set your bet amount using the controls (0 for demo mode)
2. Choose the number of mines (1-24)
3. Click "Bet" to start the game
4. Reveal tiles by clicking them
5. Cash out at any time to secure your winnings
6. Avoid mines to keep your multiplier growing!

## Auto Mode

In auto mode, the game automatically reveals tiles. You can set stop conditions:
- Stop on win
- Stop on loss
- Stop when reaching a target profit

## Technologies Used

- React 19
- Vite
- Tailwind CSS
- Modern JavaScript (ES6+)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the game in your browser.
